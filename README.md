## How to Play
open the Release Link : https://github.com/Eng-Ahmed-Farouk-backup/sabrina/releases/tag/v1.0.0
1. if you want to play it on Windows:
- download the ChillingAdventures-Windows.exe File
- Run it!
2. if you want to play it on Linux:
- download the ChillingAdventures-Linux File
- Run it!
3. if you want to play it on macOS:
- download the ChillingAdventures-macOS File
- Run it!

see the Video Here to know more about How to install and How to play the game : link will be here soon
